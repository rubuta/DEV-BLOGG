What is function?

A function is a set of statements that performs a task or calculates a value.

EXAMPLE

        
          let todoList = [
             "Go shopping",
             "Buy tobacco",
             "Do this",
             "Do that"
          ];

          function displayTodos() {

            console.log(todoList);
          
          }
          displayTodos();

The above function is used to display the lists inside the todoList
