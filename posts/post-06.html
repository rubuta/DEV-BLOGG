CONDITIONS

What is Conditions?

Conditions are statements that are created by the programmer which evaluates actions in the program and evaluates if it's true or false.

Example of simple conditions

        
          if (num > 0) {
            console.log("TRUE!");
          }
          else {
            console.log("FALSE!")
          }

When it comes to conditions they are always connected with operations(logical operations)

SOME OF THEM ARE:
>
<
>=
<=
==
===
!=
!==
